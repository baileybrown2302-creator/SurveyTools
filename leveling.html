Help & Instructions

This calculator assists with differential leveling and level loop calculations.

Starting:
  • Add Starting BM → Enter name, elevation, optional code.
  • Start New Setup → Enter BS reading + distance (direct or three-wire).
Shots:
  • Add IP → Intermediate foresight. Check "DiNi Line Leveling" if the reading is a delta from BS point elevation. Quick-repeat opens next IP automatically.
  • Add FS / Close → Foresight to TP or closing shot on known BM.
Mobile Gestures:
  • Swipe Right → Edit row
  • Swipe Left → Void/Unvoid
  • Tap Details → Expand full info
Summary Bar:
  • Shows HI during setup, Elevation between setups.
  • FS Target = distance needed to zero stadia (during setup only).
  • 📏 button → Quick stadia check without recording a point.
Stadia Warnings:
  • Amber: |Pt Stadia| > 10
  • Red: |Pt Stadia| > 25
Misclosure Accuracy:
  • After closing, click "Show Accuracy" to see classification against First/Second/Third Order standards.
